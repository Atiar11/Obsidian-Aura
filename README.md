# Obsidian Aura | The Gothic Perfume Oracle 🌑✨

> "Scent is the most powerful memory. Which shadow do you cast?"

Obsidian Aura is a high-fidelity, immersive perfume recommendation engine designed for the modern aesthetic. It combines a **Live Wallpaper UI** with an AI-driven "Oracle" to match your personality, vibe, and power with the world's most iconic fragrances.

## 🚀 Getting Started

Experience the ritual on your local machine:

### 1. Clone & Enter
```bash
git clone <your-repo-link>
cd obsidian-aura
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Ignite the Oracle
```bash
npm run dev
```
The Grimoire will be available at `http://localhost:5173` (or the port shown in your terminal).
